# Customer Behavior Analytics Platform

An end-to-end data engineering and analytics project that demonstrates how to clean raw data, architecture a relational database, and deliver executive-ready business insights.

## 🛠️ Tech Stack & Tools
* **Data Processing:** Python (Pandas, NumPy)
* **Database & ETL Engine:** PostgreSQL, pgAdmin 4, SQLAlchemy, Psycopg2
* **Business Intelligence:** Power BI Desktop (DAX, Interactive Slicers, KPI Cards)

## 🚀 Key Features
* **Automated ETL Pipeline:** Python script handles data cleaning, handling missing values, and schema formatting before executing an automatic transfer to local SQL servers.
* **Relational Data Modeling:** Created a robust, indexed SQL structure inside PostgreSQL to handle complex transactional tracking.
* **Interactive Dashboarding:** Built dynamic KPI cards with explicit DAX measures, advanced conditional formatting, and seamless multi-slicer filtering.
